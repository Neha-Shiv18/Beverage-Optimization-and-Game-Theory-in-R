# Beverage Optimization and Game Theory in R  
### ASSIGNMENT 3: SIT718  

This repository contains solutions to SIT718 Assignment 3, applying linear programming in R to beverage blending, product optimization, and game theory problems.

---

## 📌 Assignment Questions  

### Question 1: Beverage Blending Cost Optimization  
The objective of this question was to formulate and solve a linear programming model for a beverage blending problem in order to determine the optimal mix of two products that minimizes production cost while satisfying ingredient composition and demand constraints, and to analyze the sensitivity of the optimal solution to changes in cost parameters.

---

### Question 2: Maximizing Profit for a Product Mix  
The goal of this question was to maximize total profit while satisfying demand and proportion constraints for raw materials. The three seasonal products considered were Spring, Autumn, and Winter, using materials such as cotton, silk, and wool.

---

### Question 3: Game Theory Optimization Problem (Chip Game)  
The goal of this question was to model a two-player sequential game using game theory, formulate it as a payoff matrix and linear programming framework, and determine optimal strategies and expected payoffs for both players. It further involved analyzing the uniqueness of equilibrium strategies and interpreting the resulting solutions.
